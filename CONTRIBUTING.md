# contributing

thanks for helping build for the open dev.

## ways to contribute

- **code** — open a pr for ui, auth, api, search
- **profiles** — submit a project on the site (when live) or pr to `/content`
- **comparisons** — open an issue proposing a comparison page
- **docs** — improve specs in `/docs/planning`
- **sponsor** — [github sponsors](https://github.com/sponsors/shashank-sn) keeps discovery free

## before you pr

1. lowercase all ui copy (see design system)
2. only oss dependencies (see OPEN-SOURCE-STACK.md)
3. one logical change per pr
4. no paid reviews, rankings, or badge influence — ever
