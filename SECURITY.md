# security policy

## supported versions

| version | supported |
|---------|-----------|
| main | ✓ |

## reporting a vulnerability

email **security@fortheopen.dev** (or open a private security advisory on github).

please include:

- description of the vulnerability
- steps to reproduce
- impact assessment

we aim to respond within 48 hours. do not disclose publicly until we've had a chance to fix it.

## scope

- fortheopen.dev website and api
- this repository

out of scope: third-party services (github, cloudflare) — report to them directly.
